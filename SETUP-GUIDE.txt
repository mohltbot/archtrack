# ArchTrack Dashboard - Simple Setup Guide

## What This Is
Employee monitoring and project management dashboard for your uncle's architectural firm.

---

## Step 1: Download It

Open **Terminal** on your Mac mini (press Cmd+Space, type "Terminal", press Enter)

Type this and press Enter:

cd ~
git clone https://github.com/mohltbot/mission-control.git

Wait for it to finish (about 1 minute)

---

## Step 2: Install It

In the same Terminal window, type:

cd ~/mission-control/arch-firm-dashboard
npm install

Wait 5-10 minutes. It will download files.

---

## Step 3: Start The Admin Dashboard

In Terminal, type:

cd ~/mission-control/arch-firm-dashboard
npm run dev:admin

Wait until you see "Server running on port 3001"

Open **Chrome** and go to: http://localhost:5174

---

## Step 4: Start The Desktop App

Open a **new Terminal window** (Cmd+T)

cd ~/mission-control/arch-firm-dashboard
npm run dev:desktop

The employee tracking app will open automatically.
